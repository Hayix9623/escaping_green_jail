# 🏚️ 逃離方美美

**逃離方美美** 是一款以「監獄解謎逃脫」為核心的 2D 解謎遊戲。  
你被困在一座充滿綠色陰影的神秘監獄中。  
唯一的生存之道，就是不停地探索、觀察與解謎，  
找出真正能夠逃離這座牢籠的方法。

---

## 🎮 遊戲介紹

在這座詭異的監獄裡：

- 每一道機關都暗藏線索  
- 每一個房間都是謎題  
- 每一次啟動，都可能讓你更接近自由——或更深的黑暗  

你必須：

- 🧩 解開監獄中的謎題  
- 🔍 尋找並拼湊線索  
- ⚙️ 操作複雜機關  
- 🚪 找出逃脫監獄的唯一方法  

你能成功從這座綠色囚牢中逃出嗎？  
自由就在謎題的另一端。
